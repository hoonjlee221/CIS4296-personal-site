Rhythm Game Alarm Clock

This is my current individual project idea. The system combines simple rhythm games (Think PianoKeys or Guitar Hero) and sets that as the requirement to disabling the set alarm. I believe that the snooze button does more harm than good. That conclusion was made from my own personal experience waking up for work and school in the last fifteen years.

Bots/Scripts for Mobile Game Emulators

A couple months into the quarantine my friends and I got very very bored. We got into playing a bunch of mobile games like Onmyoji, Dragalia Lost, Arknights and a couple more. Since these games require a lot of recurring actions to be taken to obtain materials to progress we tried automating the process in emulators. We also used AutoHotKey to create click and repeat actions so that the game would loop player actions. We would run some of these processes overnight and not have to worry about spending too many hours during the day for the game.
